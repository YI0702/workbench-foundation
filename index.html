主流程封闭强定义 · 子环节开放共创 · 经审计后上架
①
资产晾晒

②
开放协作
官方 Skills（必备能力）+ 社区 Skills（团队成员上传），经审计后嵌入到生产/规则/洞察/排查流程任意环节

关于这个基座

四层定位：AI 对话入口（语义路由）+ 场景化 Prompt 指引（高频任务）+ 自定义视角看板（按角色看数据）+ Skills 共创区（封闭定义 + 开放共创）。 开放共建：框架由 PM 定义底稿，看板卡片 / Skills 内容预留给模块 owner 与团队成员共建。 共用基础设施：AI 对话路由 / Skills 嵌入框架 / 通用组件库 / 统一鉴权 / 数据底座 / 操作日志 —— 五个模块共享，避免重复造轮子。
